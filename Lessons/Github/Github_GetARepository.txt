Github Checkout/Pull

If you are on a new computer you may want to pull your source code from your github account.
This assumes that you already have the Github desktop application running on your computer and have sighed into Github into	
in the application.


Step 1) Open the Github desktop application.

Step 2) Select "Clone a repository"

Step 3) Select from your repository list and choose a local path where you want to put it.
	Click "Clone"

You now have the repository on your computer and can start editing, commiting and pushing the data to github.
